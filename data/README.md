# Data directory

Place **Dataset 1** train/test CSVs from Song et al. (2025) here:

- `irt_data/train.csv`
- `irt_data/test1.csv`
- `irt_data/test2.csv`

These files are large and are not committed to this repository (see the root `.gitignore`). After copying them in, run the training and evaluation steps in the main `README.md`.
