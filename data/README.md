# Data directory

**Dataset 1** train/test CSVs from Song et al. (2025) (IRT-Router) live here and are **versioned in this repository** via **Git LFS**:

- `irt_data/train.csv`
- `irt_data/test1.csv`
- `irt_data/test2.csv`

After cloning, run `git lfs pull` (or clone with `git lfs clone` / ensure Git LFS is installed) so these files are materialized locally. Then use the training and evaluation steps in the root `README.md`.
